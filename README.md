# 오금유세가(吾今有世家) 한국어 패치

**House of Legacy (吾今有世家)** 비공식 한국어 번역 패치입니다.

## 다운로드
- **다운로드 페이지: https://holk.ppit.be**
- 또는 우측 **[Releases](../../releases)** 에서 직접 받기

## 설치 방법
1. Steam에서 게임을 완전히 종료
2. zip 압축 해제 후 `House of Legacy_Data` 폴더를 **게임 설치 폴더에 덮어쓰기**
3. 게임 실행 → 설정 → 언어를 **한글** 로 변경

자세한 내용은 zip 안의 `설치방법.txt` 참고.

## 대상 버전
- 게임 표시 버전 **V0.9.03** (bundle 0.7.0)
- ※ 게임 버전이 다르면 적용하지 마세요.

## 원복
Steam → 게임 우클릭 → 속성 → 설치된 파일 → **게임 파일 무결성 확인**
