# 오금유세가(吾今有世家) 한국어 패치

**House of Legacy (吾今有世家)** 비공식 한국어 번역 패치입니다.

## 다운로드
- **다운로드 페이지: https://holk.ppit.be**
- 또는 우측 **[Releases](../../releases)** 에서 직접 받기

## 설치 방법
1. Steam에서 게임을 완전히 종료
2. zip 압축 해제 후 안의 **모든 항목**(`winhttp.dll`, `BepInEx` 폴더 등)을
   **게임 설치 폴더**(`House of Legacy.exe` 가 있는 폴더)에 복사
   - ※ 게임 원본 파일은 덮어쓰지 않고 추가만 합니다.
3. 게임 실행 → 설정 → 언어를 **한글** 로 변경

제거: 게임 폴더에서 `winhttp.dll` 과 `BepInEx` 폴더만 삭제.
자세한 내용은 zip 안의 `설치방법.txt` 참고.

## 대상 버전
- 게임 표시 버전 **V0.9.03** (bundle 0.7.0)
- ※ 게임 버전이 다르면 적용하지 마세요.

## 원복
게임 폴더에서 `winhttp.dll` 과 `BepInEx` 폴더만 삭제하면 끝입니다.
(게임 원본을 수정하지 않으므로 무결성 검사가 필요 없습니다.)
